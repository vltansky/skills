# what-i-did

Summarize your GitHub activity from the last 24 hours across all repos. Commits, PRs, reviews, and comments — presented as a standup-ready recap.

## Install

```bash
npx skills add https://github.com/vltansky/skills --skill what-i-did
```

Useful variants:

```bash
npx skills add https://github.com/vltansky/skills --skill what-i-did -g
npx skills add https://github.com/vltansky/skills --skill what-i-did -g -y
```

Or manually:
```bash
git clone https://github.com/vltansky/skills.git
cp -r skills/what-i-did ~/.claude/skills/what-i-did
```

## Prerequisites

- `gh` CLI installed and authenticated (`gh auth status`)
- Token with `repo` scope for private repo activity

## Usage

- "what did I do today"
- "my activity"
- "standup"
- "recap"
- "daily summary"
