# Skills

A collection of [Agent Skills](https://agentskills.io) for Claude Code, Cursor, Codex, and other coding agents.

## Available Skills

| Skill | Description | Install |
|-------|-------------|---------|
| [roast-my-agents-md](skills/roast-my-agents-md) | Brutally honest AGENTS.md/CLAUDE.md review backed by A/B test evidence | `npx skills add vltansky/skills/skills/roast-my-agents-md` |
| [agents-md-evals](skills/agents-md-evals) | Evaluate and optimize instruction files through controlled A/B testing | `npx skills add vltansky/skills/skills/agents-md-evals` |
| [rfc-research](skills/rfc-research) | Research a topic and produce an RFC backed by real GitHub code evidence | `npx skills add vltansky/skills/skills/rfc-research` |
| [debug](skills/debug-skill) | Runtime debugging with automated log collection — no more "check the console" | `npx skills add vltansky/skills/skills/debug-skill` |
| [chat-history](skills/chat-history-skill) | Search previous AI chat conversations from Cursor and Claude Code | `npx skills add vltansky/skills/skills/chat-history-skill` |
